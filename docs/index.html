Lightweight tray clock with NTP time

TrayClockTooltip

NTP時刻を利用できる軽量なタスクトレイ時計。標準ツールチップの代わりに
フローティング時計を表示し、Windows時計とのずれを静かに知らせます。NTP同期補助としても使えます。

小さく常駐して、
必要なときだけ知らせる

Windowsデスクトップアプリとしての存在感は控えめに。NTP取得、ログイン時の再確認、ずれ通知、時刻調整の導線だけをコンパクトにまとめています。

軽量に常駐

.NET Frameworkランタイムに依存せず、メインウィンドウを出さずにタスクトレイへ常駐します。手元環境では通常時5MB未満で動作しています。

時計を主張しないNTPチェック

普段はトレイに収まり、Windows時計とのずれだけを静かに知らせます。表示時計としてだけでなく、軽量なNTP同期チェック用にも使えます。

静かさを選べる

通知しきい値は1/3/5/7/10秒またはOffから選べます。必要以上に知らせず、設定ファイルも増やしません。

NTP基準の時計表示

Windows Timeに設定されているNTPサーバから時刻を取得し、アプリ内時計の表示に利用します。

ずれ検知と通知

Windows時計とNTP時刻のずれを、選択したしきい値で独自通知とメニューに表示します。

手動同期にも対応

予約開始や販売開始の前など、通知しきい値未満のずれでも気になるときはShift + 右クリックから時刻調整を呼び出せます。

ツールチップ風プレビュー

トレイアイコンにhoverすると、標準ツールチップの代わりに日時を中央表示したフローティング時計を表示します。

起動方法を選べる

EXEをそのまま置いて使うポータブル運用と、現在のユーザー向けに配置してスタートアップ登録する運用を選べます。

履歴をログで確認

NTP取得と時刻調整の結果をログに残すため、いつ確認し、どのくらいずれていたかをあとから見返せます。

ずれを検知したら、控えめに通知

通知はWindowsモードに合わせた独自ウィンドウです。クリックすると、必要に応じて時刻調整の確認へ進みます。

  • ✓
    起動時、ログオン時、ロック解除時にNTP時刻を確認
  • ✓
    通知しきい値は1/3/5/7/10秒またはOffから選択
  • ✓
    しきい値をOff選択でずれ通知を止め、時計と手動調整はそのまま利用可能
  • ✓
    調整時はNTP時刻を再取得してからWindows時刻へ適用
  • ✓
    時刻調整時のみUACで確認し、管理者権限では常駐しません

操作はトレイアイコンとフローティング時計だけ

普段はhoverで確認。必要なときだけ固定、移動、右クリックメニューを使います。

  1. 1
    トレイアイコンにhover

    フローティング時計をツールチップのように表示します。そのまま時計をつかんで移動できます。

  2. 2
    左クリックで固定

    時計を固定表示し、ドラッグで任意の場所へ移動できます。

  3. 3
    右クリックでメニュー

    時刻の再確認、通知しきい値の変更、時刻調整、終了を必要に応じて実行できます。Shiftを押しながら開くと、ずれ量がしきい値未満でもずれ通知を停止中でも調整でき、ログフォルダも開けます。

  4. 4
    スリープ復帰後も再確認

    ログオン/ロック解除時にもNTP時刻を再確認し、Windows時計とのずれを確認します。

高度な操作はShiftメニューへ

普段のメニューはシンプルに保ちつつ、通知設定に関係なく同期したいときやログを確認したいときだけ、Shiftを押しながらメニューを開きます。

  • ✓
    ずれ量が通知しきい値未満の場合や、通知しきい値をOffにしている場合でも時刻調整を呼び出せます
  • ✓
    ログフォルダを開き、NTP取得と時刻調整の履歴を確認できます
  • ✓
    NTP問い合わせ中は強制調整を表示せず、通常操作を邪魔しません
