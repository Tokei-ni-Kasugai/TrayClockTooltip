Lightweight tray clock with NTP time

TrayClockTooltip

NTP時刻を利用できる軽量なタスクトレイ時計。標準ツールチップの代わりに
フローティング時計を表示し、Windows時計とのずれを静かに知らせます。

小さく常駐して、
必要なときだけ知らせる。

時計アプリとしての存在感は控えめに。NTP取得、ずれ通知、時刻調整の導線だけをコンパクトにまとめています。

軽量に常駐

.NET Frameworkランタイムに依存せず、メインウィンドウを出さずにタスクトレイへ常駐します。手元環境では通常時5MB未満で動作しています。

NTP基準の時計表示

Windows Timeに設定されているNTPサーバから時刻を取得し、アプリ内時計の表示に利用します。

ツールチップ風プレビュー

トレイアイコンにhoverすると、標準ツールチップの代わりに日時を中央表示したフローティング時計を表示します。

Windowsモード追従

フローティング時計と通知は、Windowsモードに合わせてライト/ダークの見た目を切り替えます。

ずれ検知と通知

Windows時計とNTP時刻が1秒以上ずれている場合、独自通知とメニューで状態を知らせます。

手動同期にも対応

予約開始や販売開始の前など、1秒未満のずれでも気になるときはShift + 右クリックから時刻調整を呼び出せます。

起動方法を選べる

EXEをそのまま置いて使うポータブル運用と、現在のユーザー向けに配置してスタートアップ登録する運用を選べます。

履歴をログで確認

NTP取得と時刻調整の結果をログに残すため、いつ確認し、どのくらいずれていたかをあとから見返せます。

設定を増やさない

常駐時計として必要な操作はトレイメニューにまとめ、設定ファイルや常設の設定画面なしで使えるようにしています。

ずれを検知したら、控えめに通知。

通知はWindowsモードに合わせた独自ウィンドウです。クリックすると、必要に応じて時刻調整の確認へ進みます。

  • ✓
    起動時、ログオン時、ロック解除時にNTP時刻を確認
  • ✓
    1秒以上のずれがあると通知とメニューで表示
  • ✓
    調整時はNTP時刻を再取得してからWindows時刻へ適用
  • ✓
    時刻調整時のみUACで確認し、管理者権限では常駐しません

操作はトレイアイコンとフローティング時計だけ。

普段はhoverで確認。必要なときだけ固定、移動、右クリックメニューを使います。

  1. 1
    トレイアイコンにhover

    フローティング時計をツールチップのように表示します。

  2. 2
    左クリックで固定

    時計を固定表示し、ドラッグで任意の場所へ移動できます。

  3. 3
    右クリックでメニュー

    時刻の再確認、時刻調整、終了を必要に応じて実行できます。Shiftを押しながら開くと、1秒未満のずれでも調整でき、ログフォルダも開けます。

  4. 4
    スリープ復帰後も再確認

    ログオン/ロック解除時に時刻を再確認し、ずれを確認します。

高度な操作はShiftメニューへ。

普段のメニューはシンプルに保ちつつ、1秒未満でも同期したいときやログを確認したいときだけ、Shiftを押しながらメニューを開きます。

  • ✓
    既知のずれが1秒未満でも時刻調整を呼び出せます
  • ✓
    ログフォルダを開き、NTP取得と時刻調整の履歴を確認できます
  • ✓
    NTP問い合わせ中は強制調整を表示せず、通常操作を邪魔しません
